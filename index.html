Learn LOCK

This website is surposed to tell you how to program in my programming language, LOCK . It is a very simple language and was made to help people move from scratch, to text based languages, in a slightly easier way. You satart out with NUMBERS, and then move on to LOCK. sadly, LOCK is still in development.

Chapter one: Basics of NUMBERS

The console is something you need to get familiar with as it is a big part to NUMBERS. When running your code it does do some 14 second operations. This is checking if it is stable or not, and several others. When you start it will say "console has been booted correctly Code service has been booted correctly code (numbers)" then, it will tell you to write a code. Here are the codes (as of right now.) 1 = move Y 2 = move X 3 = go to a position 4 = set colour 5= set brightness 6= wait seconds. you can do these in the list order and click run!
